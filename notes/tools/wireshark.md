# Wireshark

Notes on packet capture and traffic analysis with Wireshark.

## What it's for

GUI tool for capturing and inspecting network traffic in detail — used to understand what's actually happening on the wire: protocols, handshakes, plaintext vs encrypted traffic, and anomalies.

## Getting started

- Pick the right interface to capture on (Wi-Fi vs Ethernet vs loopback) — wrong interface = no/irrelevant traffic.
- Start capture, generate some traffic (browse a site, ping something), stop capture.
